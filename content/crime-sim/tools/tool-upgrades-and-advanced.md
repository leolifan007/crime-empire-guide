The Master Key from Update 1.3 is the best late-game tool investment. It opens most standard locks instantly, dramatically speeding up heists.

## Upgrade Philosophy

Basic tools get the job done, but upgraded versions save time, reduce noise, and fail less often. Tool upgrades are not linear -- some require skill leaflets, others need specific rank milestones.

Every upgrade reduces the risk of a broken tool ending your heist prematurely.

## Tool Quality Tiers

| Tier | Example | Break Rate | Cost to Repair | Unlock |
|------|---------|-----------|----------------|--------|
| Basic | Basic Lockpick | High | ~$100-200 | Starting shop |
| Standard | Standard Lockpick | Medium | ~$200-350 | Rank 2-3 |
| Professional | Pro Lockpick Set | Low | ~$400-600 | Rank 4-5 |
| Premium | Master Key (1.3) | Very Low | ~$800+ | 1.3 content |

Higher tiers unlock larger sweet spots on lockpicking mini-games and reduce the noise generated.

## Advanced Tools Unlock Guide

### Master Key (Update 1.3)

Added in the 1.3 update, the Master Key opens most standard residential locks instantly without a mini-game. It does not work on electronic locks or high-security safes.

**Cost:** Available through 1.3 content progression. Exact price varies based on community reports.

**When to use:** Every standard door. Combined with the hacking tool for electronic locks, the Master Key covers 80% of entry points.

### Hacking Tool

Unlocked by finding the Hacking skill leaflet. Opens electronic door locks and security panels. Without it, electronic locks are impassable.

**How to get:** Complete the skill leaflet hunt. Hacking leaflets drop from special loot boxes. Keep checking the job board for houses with electronic locks -- these often contain the leaflet.

**When to use:** Houses with electronic access systems, security company buildings, late-game targets.

### Invisibility Tool

Mentioned in developer previews as an expensive but powerful tool that makes you invisible for about 60 seconds. Exact unlock requirements and cost are still being refined by the developer.

**When to use:** Emergency escapes, bypassing high-security zones, extraction from hot situations.

{{< callout "warn" >}}
The Invisibility Tool may not be available in all game versions. Check the latest patch notes on Steam before planning a build around it.
{{< /callout >}}

## Skill Leaflet Tools

Some tools require skill leaflet unlocks rather than direct purchase. These are late-game tool upgrades:

- **Advanced Lockpicking:** Unlocks professional lockpick sets
- **Advanced Hacking:** Enables the hacking tool for electronic systems
- **Tool Maintenance:** Reduces tool break rate by about 20-30%
- **Stealth Ops:** Quieter tool usage across the board

Each skill leaflet unlocks a permanent passive or active ability. Skill points are earned through leveling up.

## Investment Priority for Late Game

Once you have the basics covered, upgrade in this order:

1. **Standard Lockpick Set** -- reduces break rate significantly
2. **Pro Flashlight** -- wider beam, longer battery
3. **Advanced Hacking** -- unlocks electronic entry
4. **Master Key (Update 1.3)** -- skip most lockpicking
5. **Tool Maintenance skill** -- save money on repairs long-term

{{< insight >}}
The tool maintenance skill leaflet pays for itself after about 10 heists. If you play frequently, hunt for this leaflet before spending on expensive premium tools.
{{< /insight >}}

## Tool Storage Tips

- Keep at least one backup of each essential tool in your hideout stash
- Sell broken tools instead of repairing them if repair cost exceeds 60% of replacement
- Mark houses with electronic locks on the map -- they need the hacking tool, not lockpicks
